*** Setup

To setup these dotfiles semi-automatically, install [[https://github.com/shade-linux/soul][Soul]] and Git, clone this repo to =~/dotfiles=, and run the following command:

#+begin_src bash
  sh ~/dotfiles/setup/setup
#+end_src
